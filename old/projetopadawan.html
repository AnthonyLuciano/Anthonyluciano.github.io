ProjetoPadawan

Resumo do projeto

O projeto tem como objetivo integrar o ensino de programação nas escolas públicas para alunos do 6º ao 9º ano
com o intuito de fomentar a criatividade e habilidades tecnológicas.
Ele visa apresentar a programação como uma ferramenta transformadora e prepará-los para o futuro.
Durante dois meses, cerca de 100 crianças participaram de aulas básicas usando recursos como Python e Scratch.
A iniciativa busca modernizar a educação
promover equidade entre escolas e equipar os alunos com habilidades tecnológicas essenciais para o século XXI.
